# OpenCV Image Editor
###### Author: Jay Mody

This repo contains several image editing functions (written in c++) for OpenCV BGR Mat objects including:

- grayscale
- brightness
- contrast
- saturation
- hue
- gamma
- sharpness

**found under src/Img_edit**
